    Resources¶

    Chris Engle's Official Pages¶

    • Free Engle Matrix Games (Google Sites) — Free scenarios, original 1988 article, social matrix games online.

    • Matrix Game Links (Free Engle Matrix Games) — Curated list of professional and hobby pages.

    • The Anonymous Game (2026, online playable) — Chris Engle's latest online matrix game experiment.

    Bastionland (Chris McDowall) — OSG / RPG Perspective¶

    • Matrix Games from an RPG Perspective – BASTIONLAND — Introductory article connecting matrix games with the RPG design space.

    • Running a Matrix Game – BASTIONLAND — Hands-on AAR of the Sunrise Expansion game.

    Dreaming Dragonslayer Blog¶

    • DREAMING DRAGONSLAYER (main blog) — Known hub for Open Strategy Game discussion and design.

    • Open Strategy Games – Guiding Principles — Core principles for running OSGs.

    • More Matrix Gaming Resources and Ideas — Additional resources and variant role ideas (Consultants, Extras).

    • Landover, An Open Strategy Game — Design article for the Landover OSG scenario.

    • Landover Report – OSG Example Play — Actual play report and debrief.

    • Social Matrix Games – Review — Review of Chris Engle's Social Matrix Games book.

    • Landover OSG on itch.io — Download the Landover scenario (free/PWYW).

    PAXsims (Professional Wargaming Blog)¶

    • PAXsims — Main Site — Extensive conflict simulation and matrix games resource hub.

    • PAXsims Matrix Games Tag — All PAXsims articles tagged "matrix games."

    • MaGCK – Matrix Game Construction Kit (PAXsims page) — Overview and support materials for the MaGCK kit.

    • Toward Serious Matrix Games (2014) — Landmark article on professional use of matrix games.

    • PAXsims Games Page — Lists PAXsims-published games including MaGCK.

    Matrix Games Wiki¶

    • Matrix Games Wiki – HomePage — Community wiki for matrix game rules, design, and history.

    • Matrix Games Wiki – Links Page — Curated external links: articles, games, multimedia.

    MapSymbs (Tom Mouat's Site)¶

    • Matrix Games – MapSymbs Overview — Introduction to the Engle Matrix Game method by Tom Mouat.

    • Matrix Games Download Page – MapSymbs — Free downloadable scenarios and support documents.

    • Practical Advice on Matrix Games (PDF) – Tom Mouat — The essential practitioner's guide; widely distributed.

    History of Wargaming Project¶

    • Advanced Matrix Games for Professional Wargaming — Overview of the 2023 book by Curry, Mouat & Price, with free scenario handouts.

    • DriveThruRPG – All Hamster Press PDF Matrix Games — Full publisher listing for Engle's games.

    • MaGCK Matrix Game Construction Kit – The Game Crafter — Physical kit with rules, counters, dice, and two ready-to-play scenarios.
